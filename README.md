## 主要功能

- 设置所有网站字体样式：微软雅黑+取消加粗
- 作者实在看不下去网页上特别粗，大字重的样式
- 所以就编写这个小脚本
- 字体是附加上去的，使用时可以删除或者修改

## 注意
  此脚本会影响某些按钮的正常使用（如弹出某个div）
  遇到此情况请临时关闭脚本
